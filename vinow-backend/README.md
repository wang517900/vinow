# Vinow 越南本地生活平台 
 
## 项目介绍 
Vinow 是一个面向越南市场的本地生活服务平台，融合了团购、短视频、社区互动等功能。 
 
## 技术栈 
- 后端：FastAPI + Python 
- 数据库：Supabase (PostgreSQL) 
- 前端：Flutter/React Native (待开发) 
 
## 快速开始 
1. 配置 .env 文件中的 Supabase 信息 
2. 创建虚拟环境：python -m venv venv 
3. 激活虚拟环境：venv\Scripts\activate 
4. 安装依赖：pip install -r requirements.txt 
5. 启动服务器：uvicorn app.main:app --reload
